# 🏝️ Protaras, Cyprus - Trip Planner

A beautiful website to plan and share your 5-day adventure to Protaras, Cyprus with family and friends!

## 📋 Contents

- `index.html` - Main webpage
- `styles.css` - Styling
- `script.js` - Interactive functionality
- `images/` - Folder for trip photos

## 🖼️ Adding Your Photos

1. Take photos during your trip (or add them beforehand)
2. Save them as:
   - `images/day1-flight.jpg`
   - `images/day2-beach.jpg`
   - `images/day3-boat.jpg`
   - `images/day4-ayianapa.jpg`
   - `images/day5-dinner.jpg`
3. The website will automatically display them!

## 🌐 How to Deploy to GitHub Pages

### Step 1: Create a New Repository
1. Go to [GitHub.com](https://github.com)
2. Click **New Repository**
3. Name it: `cyprus-trip-2026` (or similar)
4. Make it **Public**
5. Click **Create Repository**

### Step 2: Upload Files to GitHub

**Option A: Using GitHub Desktop (Easiest)**
1. Install [GitHub Desktop](https://desktop.github.com/)
2. Drag this folder into GitHub Desktop
3. Commit and push to GitHub
4. Done! ✅

**Option B: Using Command Line**
```bash
cd C:\Users\morik\Desktop\Cyprus_trip_2026
git init
git add .
git commit -m "Initial commit: Cyprus trip planner"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/cyprus-trip-2026.git
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to your repository on GitHub
2. Click **Settings** → **Pages**
3. Under "Source", select **Branch: main** → **/root** → **Save**
4. Wait a few seconds...
5. Your site is now live! 🎉

Your website will be at: `https://YOUR_USERNAME.github.io/cyprus-trip-2026`

## 🎨 Customizing the Content
